## useState

- 함수를 호출한 결과가 아니라 함수 자체를 전달하면 React는 초기화 중에만 함수를 호출한다.

```js
function TodoList() {
  const [todos, seTodos] = useState(createInitialTodos());
}

function TodoList() {
  const [todos, seTodos] = useState(createInitialTodos);
}
```
